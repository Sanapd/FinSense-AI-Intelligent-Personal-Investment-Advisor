# 📊 Investment Allocator Agent (AI-Based Investment Recommendation System)

The **Investment Allocator Agent** is an AI-powered system that helps users decide how to allocate their investment budget across selected stocks using real-time market data and Large Language Model (LLM) reasoning.

This project demonstrates how **Generative AI agents**, **LangGraph workflows**, and **financial APIs** can be combined to build intelligent decision-support systems.

It is a strong **resume-ready Generative AI project** showcasing agent-based architecture.

---

# 🚀 Project Overview

This agent:

✔ Accepts investment budget from user  
✔ Fetches live stock market data  
✔ Uses LLM reasoning for allocation suggestions  
✔ Generates structured investment recommendations  
✔ Sends recommendations via email automatically  
✔ Demonstrates LangGraph agent workflow implementation  

---

# 🧠 Tech Stack

- Python

- yFinance API

# 📂 Project Structure

Investment Allocator Agent.ipynb

├── Load environment variables
├── Fetch stock market data using yFinance
├── Process investment allocation logic
├── Generate recommendations using Groq LLM
└── Send recommendation email to user

# ⚙️ Installation Guide

### Step 1: Clone Repository


git clone https://github.com/your-username/investment-allocator-agent.git


### Step 2: Navigate to Project Folder


cd investment-allocator-agent


### Step 3: Install Dependencies


pip install yfinance
pip install langchain
pip install langchain-groq
pip install langgraph
pip install python-dotenv


---

# 🔑 Environment Variables Setup

Create a `.env` file in the project root folder:


GROQ_API_KEY=your_groq_api_key

SENDER_EMAIL=your_email@gmail.com

SENDER_PASSWORD=your_email_app_password

# ▶️ How the Agent Works

Step 1: User enters investment budget  
Step 2: Agent retrieves real-time stock data  
Step 3: Groq LLM analyzes allocation strategy  
Step 4: LangGraph manages agent workflow execution  
Step 5: Final recommendation is generated  
Step 6: Email sent automatically to the user
